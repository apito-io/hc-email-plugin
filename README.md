# hc-email-plugin

SendGrid, Mailgun, and SMTP support for the Apito API platform.

## Features

- SendGrid integration
- Mailgun support
- Custom SMTP servers
- Template management
- Email analytics

## Installation

```bash
go build -o hc-email-plugin .
```

## Configuration

See `config.yml` for env vars: `EMAIL_PROVIDER`, `SENDGRID_API_KEY`, `MAILGUN_API_KEY`.

## How to Develop / Submit

See [github.com/apito-io/plugins](https://github.com/apito-io/plugins) for the CDN development guide and submission process.
